大一时候的黑框框比赛的参赛作品，一个命令行操作的魔方游戏，19年寒假编码三天的c++作品。

大概就是利用大一学的线性代数的基座标变换设计3D坐标变换算法，在CMD上拉一个句柄然后调GDI的四边形绘图函数(唯一调用的绘图API)。

命令行输入上如果乱输入可能会试出BUG，毕竟大一还不了解字典树什么的，让现在(大二)的我来写应该可以做很多改进的地方。

自我感觉还是蛮不错的一个作品吧。
